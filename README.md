# food-ordering

### Tech
Frontend: HTML, CSS, PyScript
Backend: FastAPI
Database: ZODB 

### How to run
Install necessary libraries
```
pip install fastapi uvicorn ZODB3 persistent python-dateutil
```

Run uvicorn server in backend dir
```
cd server
```

```
uvicorn main:app --reload
```
